Проект по анализу рынка видеоигр.
Описания проекта: Анализ распределения видеоигр по платформам, жанрам, регионам с 1980 по 2016гг. для определения тенденций и планирования актуальных рекламных кампаний.
Вывод: Были исследованы данные за период с 1980 по 2016 год. Составлен портрет пользователя по регионам. Проверены гипотезы о средних рейтингах. Исходя из выводов, предложена наиболее перспективная рекламная кампания.
