Анализ бизнес-показателей.
Описания проекта: анализ действий пользователей за определённый период для определения эффективности проведённых рекламных кампаний.
Вывод: Были исследованы данные о привлеченных пользователях. Проанализированы траты на рекламу по разным источникам. Построены таблицы удержания и конверсии пользователей, графики LTV, CAC, ROI.
